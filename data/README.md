
# Data Directory

This directory contains the first-party generated dataset used by the main DocVLM-Fingerprint run.

- `annotations.jsonl` defines 120 generated chart, OCR document, and scientific figure examples.
- `samples/` contains the 120 clean synthetic images referenced by `annotations.jsonl`.
- `perturbation_metadata.jsonl` defines four deterministic variants per example.
- `perturbed/` contains the 480 generated perturbation images used by the main evaluation.
- `external/` is reserved for optional real-data sanity checks and has separate licensing rules.

The `samples/` and `perturbed/` folders are intentionally kept in the public repo because they are generated by this project and make the main results reproducible without rebuilding the dataset. Imported ChartQA, DocumentVQA, or CharXiv image folders are different: those are third-party-derived assets and should remain local unless redistribution permissions are reviewed.
